Cookie Getter Tutorial

This guide helps you use the Cookie Getter extension to extract Facebook cookies for tools like this Share Tool.

  1. Download and install the Cookie Getter extension from the button below or manually from GitHub.
  2. Open Facebook in your browser and log in.
  3. Click the Cookie Getter extension icon to view and copy your cookies (e.g., c_user, xs).
  4. Paste the full cookie string into the "Facebook Cookie" field above.
  5. Use the Share Tool to share posts securely.

Note: Ensure you're logged into Facebook. For full details, check the GitHub repo.
